# COVID-19 · Statistical Analysis Dashboard

A professional-grade data visualization tool for tracking global COVID-19 trends. This application uses **Dash** and **Plotly** to provide interactive insights into infection rates, mortality statistics, and country-level comparisons.

## 🚀 Live Demo
https://covid-analysis-project.vercel.app/

## ✨ Features
* **MIT-Level UI:** Custom dark-academic theme with high-contrast Ivory text.
* **Statistical Modeling:** Integrated OLS regression and trend analysis using `statsmodels`.
* **Interactive Radar Charts:** Compare absolute metrics across different countries.
* **Responsive Tables:** Clean data presentation with `dash_table`.

## 🛠️ Tech Stack
* **Language:** Python
* **Framework:** Plotly Dash
* **Data Processing:** Pandas, NumPy
* **Analytics:** Scipy, Statsmodels
* **Deployment:** Vercel (Serverless Functions)

## 📦 Installation
1. Clone the repo: `git clone https://github.com/usmanaliclouds-jpg/Covid19-Atatistical-Analysis.git`
2. Install dependencies: `pip install -r requirements.txt`
3. Run locally: `python final_fixed.py`
